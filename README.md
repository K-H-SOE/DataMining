# Telco Customer Churn Prediction

A business intelligence project that predicts customer churn in a telecommunications company using machine learning, and translates findings into actionable retention strategies.

## About

This project applies the full BI and data mining lifecycle — from problem framing and exploratory analysis through to classification modelling, customer segmentation, and strategic recommendations with ethical reflection. Built as part of a university Data Mining module.

**Dataset:** IBM Telco Customer Churn (7,043 records, 21 features)

## Methods

Four classification models (Logistic Regression, Decision Tree, Random Forest, Gradient Boosting) evaluated using ROC-AUC and F1-score, plus K-Means clustering for customer segmentation.
